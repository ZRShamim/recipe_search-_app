## Code Flow:
For managing state I am using provider ^6.0.1. All the screens are in pages folder and UI components are inside widget folder.

```
└── lib/
    ├── pages/
    │   └── different UI pages
    ├── providers/
    │   └── state management
    ├── widgets/
    │   └── UI components
    └── styles/
        └── colors
    
```
## Feature List
```
├── Search recipe in search bar
├── Details page
└── Save recipe for later use
```
